
AI Cyberbullying Detection Chat Application

Components:
1. React Frontend
2. Node.js WebSocket Server
3. Python FastAPI AI Backend

Theme:
Black + Neon Green UI

Run Instructions:

Backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

WebSocket Server:
cd websocket-server
npm install
node server.js

Frontend:
cd frontend
npm install
npm start

Open browser:
http://localhost:3000
